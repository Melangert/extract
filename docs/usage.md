## Commands 

- `--list` - show supported formats
- `--detect <file>` - identify format without extracting
- `-o <dir>` - custom output directory
- `--verbose` - show each file as it extracts
- `--help` - show all options

## Supported formats
.zip, .tar, .tar.gz, .tgz, .tar.bz2, .tbz2, .tar.xz, .txz, .gz, .bz2, .xz
